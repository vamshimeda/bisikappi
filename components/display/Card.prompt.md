The core surface container. `raised` for product/content cards, `outline` for quiet grouping, `sunken` for wells, `dark` for bark-brown feature cards. `interactive` adds a hover lift.

```jsx
<Card variant="raised" interactive>
  <h3>Filter Coffee Blend</h3>
  <p>Chikmagalur · medium roast</p>
</Card>
<Card variant="dark">Featured this week</Card>
```
