Native select styled to match the Bisi Kaapi well, with a custom chevron. Pass `<option>`s as children.

```jsx
<Select defaultValue="m">
  <option value="s">Small</option>
  <option value="m">Medium</option>
  <option value="l">Large</option>
</Select>
```

Sizes: `sm | md | lg`. Props: `invalid`, `disabled`.
